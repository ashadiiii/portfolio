Twitter Sentiment Analysis

Overview

This project involves building a sentiment analysis model that classifies tweets as positive or negative using a machine learning pipeline. The project focuses on collecting and preprocessing Twitter data, applying Natural Language Processing (NLP) techniques, and training a Logistic Regression model to classify the sentiment of tweets. The model's performance is evaluated on a test dataset and deployed for future predictions.


Technologies Used

  • Python : Backend language for task automation and model integration.
  • Libraries:
    • Pandas and numpy: For data manipulation
    • NLTK(Natural Language ToolKit) : For stopword removal and stemming
    • skit-learn : For model development, training, and evaluation (Logistic Regression, TfidfVectorizer, train_test_split, accuracy_score).
    • Pickle: For model serialization and persistence.

Outcome

The project results in a functional sentiment analysis model capable of predicting whether a tweet is positive or negative. The Logistic Regression model, trained on the preprocessed Twitter dataset, demonstrates a high level of accuracy on both training and test sets. This model can be deployed for real-time sentiment analysis of tweets, making it applicable in various social media monitoring and analysis use cases.
