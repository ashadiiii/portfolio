Flight Management System

Overview

This project leverages the "function calling" capabilities of Large Language Models to build a functional and conversational flight management assistent. User's can interact with the system through a chat interface, requesting flight searches , bookings, updates, or cancellations.

Technologies Used

  • Streamlit : For the user interface and interactivity.
  • Python : Backend language for task automation and model integration.
  • Gemini model : To access the "function calling" capability
  • Pydantic : To ensure correct input validation
  • FastAPI: Back-end server in which the tasks are executed and information is retrieved from.
  • SQLAlChemy:Allows to interact and access the database.

Features

  • Flight Search Functionality : Users can search for flights by specifying their origin, destination, and departure date. The system retrieves available flight options from the database.
  • Real-Time Flight Booking and Flight Management :The assitant allows to book flights, update previous bookins, delete previous bookings, altogether through just the chat interface.
  • Interactive Chat Interface : The chat-based interface allows users to interact with the assistant seamlessly, receiving real-time responses and booking confirmations.
  • Conversational Feedback with Response Handling :The system analyzes the conversation to invoke relevant functions, providing a personalized and responsive user experience.
  • Scalable AI Integration : Using Googleâ€™s Vertex AI infrastructure, the system is scalable and can handle a wide range of interactions and booking management scenarios efficiently.

Outcome

This project using function calling AI offers a more efficient and beneficial flight management system compared to traditional online systems by providing natural language interaction, personalized assistance, proactive suggestions, automated task completion, and continuous learning capabilities. Users can interact conversationally, eliminating the need for complex menus and forms. The AI learns user preferences and anticipates needs, suggesting personalized options and automating complex tasks, ultimately streamlining the booking process and enhancing the overall user experience.
