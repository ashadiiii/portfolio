Flight Management System

Overview

This project involves the development of a drug recommendation system that predicts the most suitable drug for a specific medical condition based on user-described symptoms. The system leverages natural language processing (NLP) techniques and a Passive-Aggressive Classifier (PAC) model to classify conditions from user input and recommend top-rated drugs for the identified condition. The final model is deployed on a web interface using Streamlit to allow real-time recommendations.

Technologies Used

  • Python : Backend language for task automation and model integration.
  • Libraries:
    • Pandas and numpy: For data manipulation and analysis.
    • NLTK(Natural Language ToolKit) : For text preprocessing, including stopwords removal and lemmatization.
    • skit-learn : For machine learning, model training, and evaluation (Passive-Aggressive Classifier, TfidfVectorizer, train_test_split, accuracy_score).
    • Streamlit: For buildig the web-based user interface for real-time predictions and recommendations.
    • BeautifulSoup: For cleaning HTML tags from raw data.
    • Pickle: For saving and loading trained models and vectorizers.

Outcome

The Drug Recommendation System effectively recommends top-rated drugs based on symptoms provided by the user. It accurately classifies the condition and extracts the most relevant drugs based on user ratings. The integration with Streamlit allows for a real-time, interactive experience, making it highly useful for individuals seeking drug recommendations based on symptoms they describe. The system is scalable and can be extended to include more conditions and drugs for future improvements.
