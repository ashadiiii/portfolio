AI Quiz Generator

Overview

An AI-powered quiz generator that ingests PDF documents, extracts relevant information, and generates multiple-choice quiz questions on a user-specified topic. Built using a custom pipeline of natural language processing (NLP) and machine learning models to automate the creation of quizzes, this tool allows users to create tailored quizzes based on various documents in a few simple steps.

Technologies Used

  • Streamlit : For the user interface and interactivity.
  • Python : Backend language for task automation and model integration.
  • Chroma Collection : For document vectorization and storage
  • Langchain : For integrating and managing language models and document processing workflows.

Features

  • Document Ingestion : Upload PDFs and automatically ingest content into a structured format.
  • Topic-Based Quiz Generation : Users can specify the topic from the document on which they want the quiz to be based.
  • Question Customization : Select the number of questions to generate, allowing for quizzes of varying lengths.
  • Interactive Quiz Interface : Users can take the generated quiz, with immediate feedback on their answers, and view explanations for incorrect responses.
  • Quiz Navigation : Users can navigate between questions and review their answers with provided explanations.

Outcome

The AI Quiz Generator successfully streamlines the process of creating quizzes for educational or assessment purposes. It automates a traditionally time-consuming task, providing educators, students, and content creators with a quick and easy way to generate high-quality quizzes from any document. The integration of a RAG framework allows tailoring questions to users' specific interest or learning goals.
